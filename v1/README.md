# MyLinks - v1 (vibe coding)

# Input - Prompt utilizado  
Crea una aplicación web similar a linktree que tenga botones para mis redes sociales y una sección para mi información de perfil.

# Output  

# Stack generado  
- HTML5  
- CSS3 (estilos embebidos)  
- JavaScript (vanilla, sin uso real de lógica)  
- Sin frameworks o librerías externas  

# Estilos  
- Librería o CSS?  
  CSS puro (sin frameworks como Bootstrap o Tailwind)  

- Layout  
  Centrado  
  Diseño en columna  
  Uso implícito de flexbox por centrado  
  Responsive básico mediante meta viewport  

- Colores  
  Gradiente de fondo (morado → azul)  
  Botones blancos con texto oscuro  
  Esquema de pocos colores (3 principales + blanco)  

- Animaciones  
  Hover en botones  
  Escala al pasar el mouse  
  Transición suave (0.3s)

- Iconos utilizados?  
  No  

- Emojis?  
  Sí: 🚀 💻 🌎  

- Imagenes?  
  Sí: imagen de perfil placeholder  

# Lógica  
- Librería / Framework / Vanilla JS?

  Aplicación completamente estática
